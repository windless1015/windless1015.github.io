使用Pelican建立个人博客

被 nevermore - 周四, 02 8月 2018
Tags: #建站

建站记录

使用Pelican

Pelican是一个静态页面生成器,类似Jekyll和Hexo,很适合实现快速搭建博客的需要.所谓静态页面就是浏览器直接读取显示网页,无法经过服务器的编译,很适合做简单的内容展示,但是它的缺点是没有后台数据库的支持,交互性不强,不过作为个人博客是完全够用的.

windows下利用Pelican建站流程

下载必要的软件

现在做windows软件和算法开发,故这里以windows下创建作为例程.所需要的所有工具为:

  • Python3.7
  • pip
  • Git
  • Pelican
  • markdownpad2
  • MinGW

Python,git,markdownpad2,MinGW都可以直接下载(注意针对markdownpad2还需要下载一个Awesomium 1.6.6 SDK)

pip我是使用 easy_install pip

Pelican可以直接使用 pip install Pelican

创建本地博客站点

创建一个文件夹,例如为Blog,邮件使用Git Bash Here在本地使用命令行,输入pelican-quickstart.执行命令后,程序会提示输入博客的配置项,除了几个必填的其他的都可以选择默认.其实最后这些参数都可以在pelicanconf.py文件中进行修改.在Blog文件夹内会看到以下几个文件夹:content/(存放markdown或rst源文件,即你的博客的内容),output/(存放利用pelican生成的静态博客文件),pelicanconf.py是博客配置文件,主要的就是这几个,其他的可以忽略.此时在Blog目录下执行pelican content,就会在output文件夹下生成博客.这时候再进入到output目录下,执行python -m pelican.server 8081(指定端口,也可以不指定而使用默认的),这时候去浏览器执行localhost:8081就可以显示博客了.

使用主题

可以使用主题让自己的博客更加漂亮和绚丽.先从pelican的官方主题pelicanthemes挑选自己喜欢的主题,但是后来又有很多新的主题,都存放在pelican主题库上,可以去看.看好某一个主题之后,把这个主题的分支clone到本地,放置的路径最好和Blog同级,这样在pelicanconf.py里面的THEME选项便于设置.比如可以设置相对路径.

创建github个人主页

github给用户提供了个人主页,进入自己的github页面,创建一个名为username.github.io的仓库,然后在settings里面设置个人的域名,设置完毕会在仓库中生成一个CNAME的文件,这个文件很重要否则不能使用独立域名,同时github会有一个选项是Enforce https,可以勾选,你的主页就走的是https的流量,然后把output的内容推送到github上,在浏览器里面输入刚才的地址就可以看到页面效果.

给github个人主页使用独立域名

我是在name.com上购买的域名,在域名管理的地方填写github的地址(使用ping username.github.io)得到.即可实现独立域名访问个人博客.

解决谷歌字体加载过慢的问题

由于我使用的pelican主题中使用了google font的apis函数,而我大天朝访问谷歌是访问不了的.虽然我常年vpn挂着,但是速度也相对较慢.网上有几种方法:1.把谷歌字体下载下来放到github上,这种就是本地读取2.使用国内公共CDN镜像.我选的是第二种.360公共镜像,上面有怎么替换的方式.但是我看也有人说360的这个镜像库关闭了,但是我还是可以登录,可是有时候这个页面会打不开.在主题模版里面搜索fonts.googleapis.com,然后进行替换成cdn的地址即可.
